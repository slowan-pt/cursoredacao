# Politica De Privacidade

Rascunho para revisao juridica profissional.

## Dados Tratados

- cadastro;
- login;
- turma;
- pagamentos;
- arquivos enviados;
- comentarios e correcoes;
- logs tecnicos;
- notificacoes.

## Base Operacional

O tratamento deve respeitar finalidade, necessidade, seguranca, retencao proporcional e atendimento de direitos do titular.
