# Termos De Uso

Rascunho para revisao juridica profissional.

Este documento deve ser revisado antes da publicacao. Ele nao declara conformidade juridica definitiva.

## Pontos Minimos

- identificacao da plataforma;
- responsabilidades do aluno;
- responsabilidades do professor;
- regras de acesso;
- uso aceitavel;
- pagamentos;
- cancelamentos;
- propriedade dos materiais enviados;
- limitacao de responsabilidade;
- suporte;
- foro/lei aplicavel.
